# User Guide - HabiTrack

## Getting Started
1. Run the application
2. Choose from Main UI, Habits, or Progress

## Main Features

### Progress UI
- View monthly calendar with habit completion
- See completion percentages per day
- Navigate between months
- View charts and statistics

### Habit Management
- Add/edit habits
- Log daily completion
- Set habit goals

### Charts
- Pie charts for overall and per-habit completion
- Bar charts for monthly completion rates
- Streak tracking
